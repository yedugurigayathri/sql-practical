1.Write the SQL query to fetch the list of employees with same salary. 
  SELECT * FROM Worker_details 
  where Salary IN( SELECT Salary FROM Worker_Details 
  WHERE Worker_Id NOT IN ( SELECT MAX(Worker_Id) FROM Worker_Details GROUP by Salary))
